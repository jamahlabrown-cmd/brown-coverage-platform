# Brown Coverage Prototype

This is the first clickable MVP prototype for a Brown-sponsored digital insurance platform.

## What is included

- Family-friendly homepage
- Customer quote/intake flow
- Broker dashboard mockup
- Admin dashboard mockup
- Spanish landing page
- Become-a-broker page
- Demo lead routing logic
- Demo lead economics by source/quality
- LocalStorage-based lead saving for prototype testing

## How to use

Open `index.html` in a browser.

To test the flow:

1. Open `quote.html`
2. Create a demo lead
3. Open `broker.html` to see the lead in the broker dashboard
4. Open `admin.html` to see company-level metrics

## Important

This is not a live insurance quote platform. Before launch, final review is needed for:

- State licensing
- Business entity licensing
- Carrier appointments
- Marketplace consent
- Contact permission/TCPA language
- Privacy policy
- Terms of use
- Data security
- Broker agreements
- Commission/revenue share agreements
